Lotus OS - Manual

In the early times of using Lotus OS, it feels a bit hard. But once you adapt it, it adapts you.

Power Menu

Most of the Lotus OS is just same as any other Operating system expect that it's Power Menu is a bit different from any other Power menus.

  • The Sleep Button hides the Home Screen Window and shows your personal desktop (Windows, Mac, Linux etc.)
  • The Restart Button is a bit different. In other systems, the restart button restarts the computer, but in Lotus OS, restarting your OS timelapses you 2 days after that same day
  • The Shutdown button shuts down the system.

The Start Menu

The start menu is inspired by the windows 95 start menu, instead of having a seperate fram. The Start menu is literally a menu. The Start Menu has 3 sub Menus.

  • Files - Displays files inside the enviornment
  • Windows - Displays the windows opened in the enviornment
  • Apps - Displays all apps in your system
